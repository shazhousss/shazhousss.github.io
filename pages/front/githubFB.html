# vuepress开发并部署到github

参考：CSDN-Gitee(码云)、Github同时配置ssh key (opens new window)

# 一、已安装git

右键git bash here
1

# 二、输入命令

cd ~/.ssh
1

如没有.ssh文件夹，执行ssh-keygen -t rsa -C "chenchenyixin@qq.com"，把生成的.ssh文件夹中文件删掉

# 三、生成github和gitee两个平台的key

ssh-keygen -t rsa -C "chenchenyixin@qq.com" -f "github_id_rsa"
ssh-keygen -t rsa -C "chenchenyixin@qq.com" -f "gitee_id_rsa"
1
2

# 四、把public key复制到gitee和github

  1. 打开gitee_id_rsa.pub文件，将内容复制到gitee的ssh中保存：https://gitee.com/profile/sshkeys

  2. 同样，打开github_id_rsa.pub文件，将内容复制到github的ssh中保存：https://github.com/settings/keys

# 创建config文件，配置双仓库ssh信息

进入.ssh文件夹 执行命令：vi config 输入字母i，进入编辑模式 输入以下内容：

# gitee
Host gitee.com
HostName gitee.com
PreferredAuthentications publickey
IdentityFile ~/.ssh/gitee_id_rsa

# github
Host github.com
HostName github.com
PreferredAuthentications publickey
IdentityFile ~/.ssh/github_id_rsa

# 六、验证仓库ssh连接

  1. 测试gitee命令：ssh -T git@gitee.com
  2. 测试GitHub命令：ssh -T git@github.com
  3. 如果出现want to continue connecting，输入yes
  4. 如果出现successfully则表示连接成功
